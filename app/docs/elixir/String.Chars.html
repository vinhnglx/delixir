    Elixir v1.4.4 String.Chars protocol

    The String.Chars protocol is responsible for converting a structure to a binary (only if applicable). The only function required to be implemented is to_string which does the conversion.

    The to_string/1 function automatically imported by Kernel invokes this protocol. String interpolation also invokes to_string in its arguments. For example, "foo#{bar}" is the same as "foo" <> to_string(bar).
